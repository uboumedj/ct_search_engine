The Code du Travail is the French Labor Code.
This little search engine is based on the December 2023 version of the labor code, containing 20507 articles.

It is designed to search through the Code du Travail's numerous articles and display the ones most relevant to the current user query.

The logic behind the search engine is a simple BM25 (TF-IDF) scoring algorithm, written in Python.

The code for this whole project can be found on my github.
